## Como usar

1. Leia o capítulo no livro
2. Extraia o `.tar.gz` daquela semana recebido do instrutor
3. Abra o roteiro `cap-NN.md` correspondente
4. Siga os passos e entregue

## Estrutura padrão dos roteiros

Cada `cap-NN.md` segue este modelo:

- **Objetivo** — para que serve este capítulo no projeto
- **Pré-requisitos** — o que já deveria estar feito
- **O que fazer com o giropops-status** — passos específicos do projeto
- **Arquivos novos nesta semana** — o que foi adicionado ao pacote
- **Entrega** — checklist do que o aluno deve demonstrar
- **Referências no livro** — seções específicas que embasam os passos
